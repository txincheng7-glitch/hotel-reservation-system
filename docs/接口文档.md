# 酒店后端管理系统接口文档

## 1. 认证相关接口

### 1.1 用户注册
- **路径**：`/api/auth/register`
- **方法**：`POST`
- **模块/文件**：`controllers/AuthController.js`
- **功能描述**：用户注册
- **请求体**：
  ```json
  {
    "username": "testuser",
    "password": "password123",
    "email": "test@example.com",
    "phone": "13800138000",
    "role": "user"
  }
  ```
- **响应**：
  - 成功：`{"code": 201, "message": "注册成功", "data": {"id": 1, "username": "testuser", "role": "user"}}`
  - 失败：`{"code": 400, "message": "用户名已存在", "data": null}`

### 1.2 用户登录
- **路径**：`/api/auth/login`
- **方法**：`POST`
- **模块/文件**：`controllers/AuthController.js`
- **功能描述**：用户登录
- **请求体**：
  ```json
  {
    "username": "testuser",
    "password": "password123"
  }
  ```
- **响应**：
  - 成功：`{"code": 200, "message": "登录成功", "data": {"token": "jwt-token", "user": {"id": 1, "username": "testuser", "role": "user"}}}`
  - 失败：`{"code": 400, "message": "用户名或密码错误", "data": null}`

### 1.3 获取个人资料
- **路径**：`/api/auth/profile`
- **方法**：`GET`
- **模块/文件**：`controllers/AuthController.js`
- **功能描述**：获取个人资料
- **请求头**：`Authorization: Bearer jwt-token`
- **响应**：
  - 成功：`{"code": 200, "message": "获取成功", "data": {"id": 1, "username": "testuser", "email": "test@example.com", "phone": "13800138000", "role": "user"}}`
  - 失败：`{"code": 401, "message": "未授权", "data": null}`

## 2. 公开酒店相关接口

### 2.1 搜索酒店
- **路径**：`/api/hotels`
- **方法**：`GET`
- **模块/文件**：`controllers/PublicHotelController.js`
- **功能描述**：搜索酒店
- **查询参数**：
  - `city`：城市
  - `keyword`：关键词
  - `star`：星级
  - `priceMin`：最低价格
  - `priceMax`：最高价格
  - `sort`：排序方式
  - `page`：页码
  - `pageSize`：每页数量
- **响应**：
  - 成功：`{"code": 200, "message": "搜索成功", "data": {"total": 10, "page": 1, "pageSize": 10, "items": [{"id": 1, "name": "上海酒店", "star": 5, "price": 1800, ...}]}}`

### 2.2 获取酒店详情
- **路径**：`/api/hotels/{hotelId}`
- **方法**：`GET`
- **模块/文件**：`controllers/PublicHotelController.js`
- **功能描述**：获取酒店详情
- **路径参数**：
  - `hotelId`：酒店ID
- **查询参数**：
  - `checkIn`：入住日期（可选）
  - `checkOut`：退房日期（可选）
- **响应**：
  - 成功：`{"code": 200, "message": "获取成功", "data": {"id": 1, "name": "上海酒店", "address": "黄浦区中山东一路1号", "description": "酒店描述...", "star": 5, "images": [...], "rooms": [...]}}`
  - 失败：`{"code": 404, "message": "酒店不存在", "data": null}`

### 2.3 获取价格日历
- **路径**：`/api/hotels/{hotelId}/price-calendar`
- **方法**：`GET`
- **模块/文件**：`controllers/PublicHotelController.js`
- **功能描述**：获取价格日历
- **路径参数**：
  - `hotelId`：酒店ID
- **查询参数**：
  - `yearMonth`：年月（格式：YYYY-MM）
- **响应**：
  - 成功：`{"code": 200, "message": "获取成功", "data": [{"date": "2026-03-01", "price": 1800, "available": 10}, ...]}`

## 3. 商家相关接口

### 3.1 创建酒店
- **路径**：`/api/merchant/hotels`
- **方法**：`POST`
- **模块/文件**：`controllers/MerchantController.js`
- **功能描述**：创建酒店
- **请求头**：`Authorization: Bearer jwt-token`
- **请求体**：
  ```json
  {
    "name": "上海酒店",
    "address": "黄浦区中山东一路1号",
    "description": "酒店描述...",
    "star": 5,
    "openingDate": "2018-06-01",
    "images": ["https://.../main.jpg"],
    "facilityIds": [1, 2],
    "tagIds": [1, 2]
  }
  ```
- **响应**：
  - 成功：`{"code": 201, "message": "创建成功", "data": {"id": 1, "status": "pending", "createdAt": "2026-02-21T09:56:10Z"}}`
  - 失败：`{"code": 400, "message": "酒店名称和地址不能为空", "data": null}`

## 8. 响应格式说明

所有API接口的响应格式统一为：

```json
{
  "code": 200,       // 状态码
  "message": "成功", // 响应消息
  "data": {}         // 响应数据
}
```

### 状态码说明

| 状态码 | 含义 |
|--------|------|
| 200 | 请求成功 |
| 201 | 创建成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
